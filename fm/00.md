# No.0 Web Worker 有名字了！

## 描述

No.0 Web Worker 有名字了！
主播：辛宝 Otto、刘威 Franky。

挖新坑了！这次我们来尝试搞一个有关程序员的音频播客！作为第 0 期节目，当然是要起名字、规划节目了。

《Web Worker》是两个前端程序员闲聊的音频播客节目。节目将围绕程序员领域来瞎聊，聊资讯、聊职场、聊技术选型...... 只要是和 web 开发有关的都可以聊。因为两个主播是前端程序员目前会以前端为视角切入。
